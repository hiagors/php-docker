# php-docker

## :rocket: Começando
``` bash
  # Clonar o projeto:
  $ git clone https://github.com/hiagors/php-docker.git

  # Entrar no diretório:
  $ cd php
```

## :gear: Executando docker-compose
```bash
  # Entrar no diretório do repositório:
  $ cd php

  # Executar o docker compose:
  $ docker-compose up

  # Rodar o container docker:
  $ docker run php

  # Abrir a aplicação web:
  $ localhost
```
